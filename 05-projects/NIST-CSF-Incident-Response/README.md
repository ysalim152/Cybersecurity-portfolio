## 📌 Overview
Simulation of a **Denial of Service (DoS) ICMP flood attack** analyzed using the **NIST Cybersecurity Framework (CSF)**.

## 🎯 Objectives
- Analyze cybersecurity incident
- Identify root cause
- Apply NIST CSF
- Recommend mitigation strategies

---

## 🧠 NIST CSF Phases
- Identify
- Protect
- Detect
- Respond
- Recover

Attacker → Internet → Firewall → Internal Network

- SOC Documentation

Improved understanding of DoS attacks and structured incident response using NIST CSF.
