# Overview

This project was developed as part of a cybersecurity portfolio to demonstrate practical knowledge of incident response, network security, risk assessment, and security hardening using the **NIST Cybersecurity Framework (CSF)**.

The project is based on a simulated **Denial-of-Service (DoS)** incident involving an ICMP Flood attack against an organization's internal network.

The methodologies, terminology, and security recommendations presented in this repository are based on industry-recognized standards and best practices.
